Projects

Password Manager

A simple master-password based password manager that stores encrypted passwords in a JSON file. The program uses the AES encryption algorithm to encrypt and decrypt passwords. The program is written in C++ and uses the OpenSSL library for encryption/decryption and nlohmann/json for handling JSON files.

PNG Steganography

A simple program written in C, with the help of zlib, that hides text in a PNG image. The program uses the Least Significant Bit (LSB) technique to hide content in the image's pixel data. This project is currently undergoing a full rewrite.

HTTP/1.1 Server

A simple HTTP/1.1 server written in C++ that serves static files. The server supports GET requests and can server HTML and PNG image file types (AFAIK) in its current state.

DNS Proxy

A simple DNS proxy server written in C++ that either blocks or forwards requests to a specified DNS server. The proxy parses requests from clients and checks the domain name against a blacklist. If a domain name is in the blacklist the query is modified before it is sent to the selected DNS server.

CMake Project Generator

A simple C++ terminal program that generates a CMake project structure for C++ projects. The script creates a CMakeLists.txt file and a src directory for source files. The program uses C++17's std::filesystem and nlohmann/json.
